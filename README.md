# PureRate Web

This repository contains the landing page for **PureRate**, a lightweight macOS Menu Bar application that effortlessly synchronizes your system's hardware audio sample rate to match Apple Music.

## Features
- Modern Glassmorphism UI
- Responsive design
- SEO optimized
- Built with vanilla HTML, CSS, and JS

## Deployment
You can deploy this website to GitHub Pages, Vercel, or any static hosting service.
To preview locally:
```bash
open index.html
```

## Structure
- `index.html`: Main landing page
- `style.css`: Design system and components
- `main.js`: Interactive elements
- `images/`: High-resolution visual assets
